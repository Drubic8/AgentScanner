# VNish Firmware API Reference

**Официальная документация:** Swagger UI (Сохранено локально в проекте `SwaggerUI.html` / `SwaggerUI.mhtml`).
**Тип API:** Современный REST API (JSON).

## 1. Архитектура
Кастомная прошивка VNish полностью заменяет стандартный CGMiner на собственный демон. Они предоставляют красивый, задокументированный REST API.
В отличие от стоковых прошивок, VNish не прячет данные по `.cgi` скриптам, а использует структурированные роуты (обычно `/api/v1/...` по порту 80 или 443).
