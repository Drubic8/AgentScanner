# Whatsminer (MicroBT) API Reference

**Официальная документация:** [https://apidoc.whatsminer.com/](https://apidoc.whatsminer.com/)
**Главный порт:** `4433` (Специальный бинарно-JSON протокол).
*Ограничения:* Максимум 10 одновременных клиентов. 
*Heartbeat-пакет:* `0x00, 0x00, 0x00, 0x00`.

## 1. Структура пакета
Общение идет не чистым текстом, а пакетами с заголовком:
* Первые 4 байта (Little-Endian) — длина JSON-строки.
* Далее идет сама JSON-строка.

## 2. Чтение статистики (API v3)
В новых прошивках (API v3.x) используются расширенные команды для получения данных:
* `{"cmd": "get.device.info"}` — Аппаратные данные, модель, MAC, сетевые настройки и коды ошибок.
* `{"cmd": "get.miner.status", "param": "summary"}` — Хешрейт, аптайм, температуры, обороты кулеров.
* `{"cmd": "get.miner.status", "param": "pools"}` — Текущие настройки и статус пулов.
*Чтение (команды `get.*`) включено по умолчанию и не требует авторизации*.

## 3. Авторизация и Управление
Для отправки управляющих команд (`set.*`) действуют строгие правила безопасности:
1. **Предварительная настройка:** Заводской пароль `admin` должен быть изменен (с ним запись блокируется). В утилите WhatsminerTool должен быть активирован переключатель "API write access".
2. Запросить `salt` (соль) через `{"cmd": "get.device.info"}`.
3. Сформировать токен, используя пароль (для изменения системных настроек нужен уровень `super`).
4. Подписать команду алгоритмом HMAC-SHA256.

**Основные команды управления:**
* **Reboot (Перезагрузка):** `{"cmd": "set.system.reboot"}`
* **LED Blink (Поиск):** `{"cmd": "set.system.led", "params": [{"color": "red", "period": 200, "duration": 100...}]}`
* **Sleep (Остановка майнинга):** `{"cmd": "set.miner.service", "param": "stop"}` 
* **Resume (Возобновление майнинга):** `{"cmd": "set.miner.service", "param": "start"}`

## 4. Административные команды (Управление доступом)
* **Смена пароля:** `{"cmd": "set.user.change_passwd", "param": "<Base64_Encrypted_String>"}`. Параметр должен содержать зашифрованные старый и новый пароли.
* **Права пользователей:** `{"cmd": "set.user.permission", "param": {"user": "user1", "permission": "set.system.led,set.miner.pools"}}`. Позволяет `super`-администратору выдавать гранулярные права (через запятую) другим аккаунтам.

## 5. Определение статуса Sleep
В официальном API нет параметра `Status: Sleep`. Программа-сканер вычисляет этот статус косвенно:
* Если в ответе `summary` есть сообщение `"btminer process is down err"` (старые прошивки).
* Если `working` = `false`, хешрейт = 0, нет аппаратных ошибок, а `uptime` (elapsed) > 60 секунд (новые прошивки).
