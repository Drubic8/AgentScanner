# Whatsminer (MicroBT) API Reference

**Официальная документация:** [https://apidoc.whatsminer.com/](https://apidoc.whatsminer.com/)
**Главный порт:** `4433` (Специальный бинарно-JSON протокол)

## 1. Структура пакета
Общение идет не чистым текстом, а пакетами с заголовком:
* Первые 4 байта (Little-Endian) — длина JSON-строки.
* Далее идет сама JSON-строка.

## 2. Чтение статистики
* **Команда:** `{"cmd": "get.psu.hashrate"}` (или `get.miner.hashrate`).
* Чтение происходит без пароля.

## 3. Авторизация и Управление (API v3)
Для отправки команд (Перезагрузка, Светодиод, Смена пулов) требуется сложная система подписи:
1. Запросить `salt` (соль) через `{"cmd": "get.device.info"}`.
2. Сформировать токен, используя пароль (по умолчанию `admin` или `super`, для изменения настроек нужен `super`).
3. Подписать команду HMAC-SHA256.

* **Reboot:** `{"cmd": "set.system.reboot"}`
* **LED Blink:** `{"cmd": "set.system.led", "params": [{"color": "red", "period": 200, "duration": 100...}]}`
